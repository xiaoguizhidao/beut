INSTALLATION:

NOTE: To manually install Magic Zoom Plus please read 'readme_manual.txt'.

IMPORTANT: If you use Magento Compiler, disable it. After installation is complete, go to Magento Compiler and run the compilation process.

1. Unzip the contents of the zip file and upload them to your Magento directory.

2. Go to http://your.site.url/magiczoomplus/ in your browser and follow the instructions.

3. Go to 'System -> Cache Management' and refresh the cache.

4. Configure the Magic Zoom Plus module through the 'Magic Toolbox -> Magic Zoom Plus settings' menu in the Magento admin panel.
